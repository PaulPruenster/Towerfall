# Towerfall

A towerfall style game hopefully with wii remote support

## Project Layout

- `assets/`
  Art, imported packs, and generated gameplay textures.
- `music/`
  Background music tracks.
- `scenes/actors/`
  Reusable actor scenes such as the player and arrows.
- `scenes/audio/`
  Audio-focused scenes such as the song player.
- `scenes/effects/`
  One-shot visual effects like explosions.
- `scenes/gameplay/`
  Reusable gameplay objects such as chests, gates, jump pads, hazards, and platforms.
- `scenes/levels/`
  Playable level scenes.
- `scenes/ui/`
  Main menu and UI entry scenes.
- `scripts/actors/`
  Actor logic for players and projectiles.
- `scripts/audio/`
  Audio playback logic.
- `scripts/gameplay/`
  Gameplay object and world logic.
- `scripts/systems/`
  Shared support systems such as procedural SFX.
- `scripts/ui/`
  Menu/UI scripts.
